# 📊 Amazon Sales Analytics Dashboard

## 📌 Overview

This project analyzes Amazon sales data using **MySQL, SQL, Power BI, and DAX** to uncover sales trends, product performance, customer insights, and fulfillment efficiency. The project demonstrates the complete data analytics workflow from data cleaning and SQL analysis to interactive dashboard development.

---

## 🛠️ Tech Stack

- Power BI
- MySQL
- SQL
- DAX
- Excel

---

## 📂 Dataset

- Amazon Sales Dataset
- 120K+ Orders
- 31 Columns

Due to GitHub file size limitations, the dataset is provided as a compressed ZIP file.

File:
- Amazon_Sales_Dataset.zip

# 🗄️ SQL Analysis

The project includes SQL queries for:

- Total Revenue
- Total Orders
- Monthly Sales
- Revenue by Category
- Revenue by State
- Revenue by City
- Top Selling Products
- Order Status Analysis
- Window Functions (RANK, DENSE_RANK, LAG, LEAD)
- Common Table Expressions (CTEs)

# 📈 Key Business Insights

- Identified top-performing product categories and SKUs.
- Analyzed monthly revenue trends and regional sales distribution.
- Evaluated fulfillment performance and order status.
- Compared Amazon and Merchant fulfillment methods.
- Analyzed customer purchasing behavior and B2B orders.
